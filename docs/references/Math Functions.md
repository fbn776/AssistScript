## abs
`absolute`<br/><br/>
**Description:** <br/>Returns the absolute value of the given number.<br/>

**Syntax:**<br/>

`abs <number>`<br/>


**Example:**<br/>

```asrc
abs -5
```

**Result:**<br/>

```asrc
5
```

## round
**Description:** <br/>Rounds the given number to the nearest integer.<br/>

**Syntax:**<br/>

`round <number>`<br/>


**Example:**<br/>

```asrc
round 3.14
```

**Result:**<br/>

```asrc
3
```

## sign
**Description:** <br/>Returns the sign of the given number.
Returns 1 if positive else -1.
For positive 0 => +0
For negative 0 => -0<br/>

**Syntax:**<br/>

`sign <number>`<br/>


**Example:**<br/>

```asrc
sign -5
```

**Result:**<br/>

```asrc
-1
```
