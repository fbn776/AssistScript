## random
`rand` `randf`<br/><br/>
**Description:** <br/>Returns a random float between the specified ranges (optional). If no range is given then random float b/w 0 and 1, if one is specified number b/w 0 and that num.<br/>

**Syntax:**<br/>

`random <min?> <max?>`<br/>


**Example:**<br/>

```asrc
random 5 10
```

**Result:**<br/>

```asrc
8.338382535573796
```

