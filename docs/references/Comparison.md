## lt
`less-than`<br/><br/>
**Description:** <br/>Checks if the first number is less than the second.<br/>

**Syntax:**<br/>

`lt <number> <number>`<br/>


**Example:**<br/>

```asrc
lt 2 3
```

**Result:**<br/>

```asrc
true
```
